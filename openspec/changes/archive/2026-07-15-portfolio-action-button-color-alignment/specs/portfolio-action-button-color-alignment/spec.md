## ADDED Requirements

### Requirement: Portfolio action buttons use active-tab color
The portfolio tab action buttons SHALL use the same solid accent color as the active tab state instead of gradient styling. The button text SHALL remain legible against the solid accent background.

#### Scenario: Add and remove buttons match active tabs
- **WHEN** the portfolio tab renders the Add and Remove action buttons
- **THEN** each button SHALL use the same solid accent color as the active tab
- **AND** the buttons SHALL NOT use a gradient background
